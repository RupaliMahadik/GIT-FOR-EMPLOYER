Welcome

Welcome to the data portal of the National Center for Biomedical Glycomics. This webpage provides the data that has been generated analysis cell lines that have been differentiated into different types of tissue.The data that can be found is

  • Whole cell glycan expression data (PMID: 27993942)
  • Whole cell glyco-protein expression data (PMID: 27993942)
  • Cell surface glyco-protein expression data (PMID: 27993942)
  • Reporter glyco-protein expression data (PMID: 27993942)
  • Secretome glycol-protein expression data (PMID: 27993942)

Data can be browsed by selecting a single data sets below or by selecting multiple datasets for comparison
